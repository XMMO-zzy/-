## 3. 核心协作约定

- 接口冻结：数据库字段、阈值、API字段禁止私改
- 变更流程：评审 -> 实现 -> 回归 -> 升版
- 每日节奏：站会 15 分钟，更新问题清单
- 每周节奏：周二/周四联调，周五集成演示与门禁验收
